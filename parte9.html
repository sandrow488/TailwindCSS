Más utilidades de Tailwind CSS

2) Tipografía: tamaño, peso, alineación, espaciado

  • text-sm, text-lg, text-4xl: escala de tamaños.
  • font-medium, font-bold: peso del texto.
  • leading-snug, tracking-wide: altura de línea y espaciado entre caracteres.
  • text-center, text-left: alineación.
<div class="p-6 bg-gray-100 rounded-lg">
  <h1 class="text-3xl font-bold text-gray-800">Bienvenido a Tailwind CSS</h1>
  <p class="mt-4 text-gray-600 leading-relaxed">
    Este párrafo tiene un interlineado relajado y un color suave para mejorar la legibilidad.
  </p>
  <p class="mt-2 text-sm font-medium text-gray-500 tracking-wide">
    Todo se controla con clases: tamaño, color, peso y espaciado.
  </p>
</div>

Comentarios:

  • leading-relaxed hace que el texto “respire” más.
  • La escala de tipografías está lista para producir títulos efectivos, párrafos cómodos y subtítulos.

3) Colores: fondo, texto, borde, nivel de opacidad

Tailwind usa una paleta basada en tonos predefinidos (como slate, gray, blue, red) numerados de 50 a 950.

  • bg-blue-500 → azul medio.
  • text-gray-700 → gris oscuro.
  • border-gray-200
  • bg-red-500/80 → con 80% de opacidad, usando la sintaxis del “slash”.
<div
  class="p-4 bg-white border border-gray-200
         dark:bg-slate-800 dark:text-gray-200 dark:border-gray-700
         rounded-lg"
>
  <p class="text-gray-700 dark:text-gray-300">
    Este texto cambia en modo oscuro.
  </p>
  <button
    class="mt-4 px-3 py-2 bg-rose-500/90 hover:bg-rose-600 text-white rounded-md"
  >
    Eliminar
  </button>
</div>

Comentarios:

  • bg-rose-500/90 aplica opacidad sin necesitar rgba().
  • El bloque cambia fondo, borde y texto automáticamente con variantes dark:.
  • Puedes usar valores arbitrarios como bg-[#316ff6] si necesitas un color específico.

Ejercicio guiado — Guion editorial con jerarquía, ritmo y contraste

Redacta un fragmento editorial aplicando una escala tipográfica coherente, buen ritmo de línea y un uso medido del color. El objetivo es que el texto se lea con fluidez y tenga una voz visual consistente sin escribir CSS adicional.

Requisitos mínimos

  • Usar una escala que incluya al menos: text-xs, text-base, text-2xl o text-4xl.
  • Controlar legibilidad con leading-relaxed o leading-7, y limitar el ancho con max-w-prose.
  • Aplicar acentos cromáticos discretos con text-brand y fondos sutiles (bg-*-500/10).
  • Incluir al menos un bloque semántico: blockquote, figure con figcaption o un callout.
  • Compatibilidad con modo oscuro usando variantes dark:.

Pasos guiados

  1. Define un contenedor de lectura con max-w-prose y separaciones verticales usando space-y-4 o space-y-6.
  2. Crea la jerarquía: etiqueta editorial en text-xs y uppercase con tracking amplio; titular en text-4xl y leading-tight; cuerpo en text-base con leading-relaxed.
  3. Añade un subtítulo o bajada en text-lg con color atenuado para diferenciarlo del cuerpo.
  4. Incluye un blockquote o figure con figcaption; realza con borde o fondo suave.
  5. Destaca una palabra o frase clave con bg-rose-500/10 o bg-brand/10 y rounded para crear un acento sin romper el flujo.

Editorial

Diseña una voz consistente

Una tipografía equilibrada y una paleta sobria facilitan la lectura y proyectan confianza.

Empieza por establecer una escala clara entre título, bajada y cuerpo. Mantén el ancho de línea acotado con medidas de lectura cómodas y controla el ritmo con una altura de línea generosa. Usa el color como guía, no como protagonista.

Consejo: limita los estilos a unas pocas decisiones fuertes (familia, escala, interlínea, acentos) y repítelas con consistencia en todo el documento.

La tipografía clara y los colores coherentes comunican profesionalismo incluso en proyectos sencillos.
  • Jerarquía: diferencia clara entre niveles (xs, base, 2xl/4xl).
  • Ritmo: usa leading-relaxed para mejorar la legibilidad.
  • Acentos: text-brand y fondos suaves para enfatizar.

Checklist

  • ¿El titular destaca sin romper el ritmo del resto del texto?
  • ¿El ancho de texto está limitado con max-w-prose y la lectura es fluida?
  • ¿Los acentos cromáticos son sutiles y mantienen buen contraste en claro/oscuro?
  • ¿Has usado al menos un bloque semántico (blockquote, figure+figcaption o callout)?

Retos opcionales

  • Añade una “bajada” secundaria con text-base y text-gray-600 para modular tono.
  • Introduce una nota al pie con text-xs y tracking-wide para diferenciarla del cuerpo.
  • Explora un acento alternativo con bg-amber-500/10 y comprueba el contraste en oscuro.
